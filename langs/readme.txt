Правила:

1. Кодировка файлов - UTF8, используется для поддержки национальных символов.
   Для правки файлов рекомендуется использовать редактор Notepad++
   (скачать можно по адресу http://notepad-plus-plus.org/release/5.9),
   т.к. он автоматически распознает файлы в кодировке UTF8,
   корректно открывает и сохраняет.
2. Первая строка файлов lng - заголовок с обозначением кода и названия языка,
   быть с ней внимательными чтобы не повредить. Знак ";" в начале - обязателен.
3. Порядок строк в файле не менять.
4. Дублирующиеся строки оставлять как есть.
5. Обозначения вида %s или %n не трогать - это места подстановки других строк
   или чисел.
6. Файл russian.sample содержит эталонный русский текст. Можно вставить его
   содержимое, начиная со второй строки, в файл иного языка и построчно 
   переводить. Эталонный файл не следует переименовывать в lng-файл,
   т.к. русский язык вшит в программу.
7. Строки вида <reserved> - зарезервированы, переводить не нужно.
8. Перевод можно тестировать на любом этапе. Данные из файла загружаются
   при его выборе, поэтому можно при запущенной программе периодически
   менять выбранный язык и просматривать изменения в интерфейсе.
9. При выходе новых версий программы в этом файле будут содержаться номера
   изменившихся или новых строк, которые нужно обновить в переводах.

Внимание: файл английского языка - тестовый, поэтому участие в его грамотном 
переводе - приветствуется. Однако в первую очередь интересуют переводы на
языки славянской группы или республик бывшего СССР.

Каждый участник перевода на другие языки будет вписан в справку как
автор своих переводов. Впишите своё имя в летопись программы :)
