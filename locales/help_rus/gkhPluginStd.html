Стандартные плагины

В стандартной комплектации программы поставляется ряд плагинов:

  • Калькулятор выражений, стандартный сервисный плагин;
  • Календарь, стандартный сервисный плагин;
  • Справочник имен, стандартный сервисный плагин;
  • Линия времени, стандартный сервисный плагин;
  • Поточный ввод, плагин для непрерывного облегченного ввода данных из источников;
  • Импорт росписей, импорт данных из сторонних текстовых росписей;
  • Просмотр изображений, просмотр изображений для облегчения работы со сканами или фотоснимками источников;
  • Полнотекстовый поиск, специализированный поиск по базе данных, эмуляция поиска по тексту как в интернет-поисковиках;
  • 3D Визуализация (экспериментальный), просмотр трехмерных диаграмм деревьев;
  • Игра "Жизнь Конвея", мини-игра для релаксации.
