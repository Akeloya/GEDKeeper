Записи групп

Группы предназначены для удобства навигации, это объединения людей по некоторому общему социальному признаку.

Для примера, удобно объединить в группу "Участники Великой Отечественной войны" всех соответствующих родственников. Или выделить отдельную группу "Корреспонденты" - для тех, с кем ведется переписка.

Включать персональные записи в группу можно двумя способами:

  • кнопкой "Добавить" на закладке "Участники" диалога редактирования группы;
  • кнопкой "Добавить" на закладке "Группы" диалога редактирования персоны.

Исключать персоны из групп можно полностью аналогичным способом с помощью кнопки "Удалить" на тех-же закладках.

Все имеющиеся в базе данных группы доступны через единую таблицу на закладке "Группы" основного рабочего окна. Справа от таблицы - в сводке по записи, можно просмотреть информацию по группе, а также список всех других людей, которые в неё входят.

Также смотрите: Персоны, Заметки, Мультимедиа-материалы.
