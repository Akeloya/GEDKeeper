Инструменты: Поиск патриархов

Данный инструмент предназначен для автоматического поиска наиболее глубоких предков в каждой из исследуемых линий родства. Программа просматривает различные ветви базы данных и выбирает в каждой мужчину, у которого неизвестны родители и число поколений потомков соответствует заданному. В случае, если у данного предка не задан год рождения - такой год расчитывается автоматически исходя из года рождения ближайших потомков. Такой приблизительный расчет служит только для информативных целей. Помимо количества поколений, для каждого патриарха определяется количество потомков.

Для начала работы с этим инструментом, нужно определиться - какой длины нужно искать ветви. К примеру, если интересуют все патриархи, у которых не менее 3 поколений потомков - нужно в поле "Поколений потомков не менее" ввести требуемое количество. Затем нажать кнопку "Поиск". Поиск покажет всех патриархов, соответствующих критерию отбора.

Признак "Не учитывать даты", укажет программе, что расчетная дата рождения патриарха не нужна. В этом случае, у патриархов без явно заданной даты рождения будет год будет равен "-1".

Обозначение [*] перед именем патриарха показывает, что для этого человека установлен признак-закладка "Патриарх" (см. Метки индивидуальных записей).

Кнопка "Установить признак" служит для установки закладки "Патриарх" не переходя из данного инструмента в диалог правки персональной записи.

Кнопка "Диаграмма" вызовет т.н. "граф взаимосвязей между патриархами". Эта диаграмма - сильно специфический инструмент, и на первый взгляд сложна для восприятия. Рекомендуется начинать работу с этой диаграммой, выставив предварительно достаточно большое число поколений потомков у патриархов - это снизит количество узлов в графе связей. На диаграмме, после её построения и стабилизации внешнего представления, будут видны узлы самих патриархов (темно-синий цвет узлов) и узлы пересечений ветвей их потомков (сиреневый цвет узлов). Цифровые подписи узлов - это идентификаторы семей. Если навести курсор на любой узел - появится подсказка с названием семьи в данном узле. Стрелки показывают направление от семей предков к семье потомков, в которых два рода пересекаются.
