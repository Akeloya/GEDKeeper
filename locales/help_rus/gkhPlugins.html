Плагины

Эта программа поддерживает два типа расширения: написание скриптов на языке Lua и написание плагинов на языках C# или VB.NET.

Как правило поддержка плагинов добавляется в приложения по следующим причинам:

  • для расширения функций приложения без необходимости пересборки и распространения всем пользователям;
  • для добавления функционала без необходимости доступа к оригинальному исходному коду приложения.

Для создания плагинов, требуется использовать общую библиотеку ядра программы GKCore.
