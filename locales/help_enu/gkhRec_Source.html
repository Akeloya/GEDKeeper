Source record

A source is a mentioning any private document or file from archive fund, where you found some information about persons. Source records are used for the following purposes: they store specific information about information's origin, unite and simplify search for person, family and event, that were mentioned in one document.

This can be a birth or death certificate, audit tale, register of births, confessional book and other documents.

Each source reference has two additional properties: page or sheet number, where the mentioning was made, and authenticity level for the information. Having page or sheet number is helpful when you need to review the source once again. Authenticity level quickly shows you how much you can trust a source of a particular information. When a source document has blots, damages and you could not have read text with assurance, you should assign "Questionable reliability of evidence" level to such source. This can prevent you from false conclusions.

Authenticity level also affects authenticity index, which is calculated for a person in statistics module to analyze mean values. Tree diagrams use authenticity level to make shown information more obvious.

Source also has archive property that defines where the source is stored. It is worth noting that archive here is not necessary a state archive or private archive institution, but it is also a private document collection or other specific document storage.

There is a uniting table on the "Sources" tab of the main working window, which includes all sources available in database. Source summary to the right of the table shows all information about selected seource and links made from other records to this one.
