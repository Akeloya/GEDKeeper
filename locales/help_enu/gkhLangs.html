Localization

GEDKeeper has multilanguage support. You can use, for example, Russian localization or another one at your option. You can change user interface localization using "Service\Options" on the "Common" tab. Interface language is changed immediately -- you do not have to restart the application.

Also you can contribute to the project by translating user interface into your native language. Just translate the most suitable for the user "lng" file from the "locales" project subdirectory and email the translated file to author of GEDKeeper. After checking, your file will be included in the next GEDKeeper release.
