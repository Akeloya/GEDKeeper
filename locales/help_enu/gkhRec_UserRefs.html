User reference

User reference/footnote is a special data type used in person records to add various notes. The standard does not define application for this object and it was introduced only for user comfort, to be used as short note.

But GEDKeeper itself defines the application for user references: to show special marks on persons in tree diagrams. For example, if a man waged The Great Patriotic War and his/her person record has the respective reference, then this man is shown in tree diagram with red star. If a man died, respective reference adds black edge to the star.

There are following predefined user references available in GEDKeeper:

 USSR:WW II:Rear worker
 USSR:WW II:Combatant
 USSR:WW II:Killed in action
 RE:George knight
 USSR:Victimizated
 Religion:Islam
 Religion:Catholicism
 Religion:Orthodoxy
 Religion:Old Belief

New user references will be added to the program as needed.

You change user references on "Reference/footnote" tab of person editor window.
