Types of database records

GEDKeeper stores database records of the following types:

  • Person
  • Family
  • Note (text attached to a database record, piece of document, comment)
  • Multimedia content (reference to document, photo, audio and video recording)
  • Source (private document and document from archive fund)
  • Archive (state archive and private one)
  • Group (group of people having common social attributes)
  • Research (main directions of investigation and priority of those directions)
  • Task (discrete investigation task)
  • Correspondence (information about correspondence and communication with relatives and other researchers and archive organizations)
  • Location (geographical location and location coordinates)

Main working window has a dedicated tab for each record type:

The following operations are available for all record types: add, modify, remove and filter.

Each tab has two visual parts: table with record list on the left and summary window that shows detailed information about selected entry in the table.

You can order entries in the table by clicking a column - a property of record. To avoid incidental changes, you cannot change data directly in the table (in-place editing is disabled). To open editor window for a record you should double click the row, or select a row and press [ Ctrl + Enter ] key combination.

All record types have two common attributes: record identifier and last modification time ("#" and "Changed" columns respectively). All other attributes in a record type are record-specific.

You can activate filter on the table using button on toolbar, or use "Edit\Filter" menu item, or press [ Ctrl + F ] keys. In addition, the "Persons" tab has quick search; use "Edit\Search" menu item.
