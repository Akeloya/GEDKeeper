Family tree

There are three types of tree diagrams GEDKeeper can create: tree of ancestors (ascending relationship tree), tree of descendants (descending relationship tree) and combined tree -- tree of ancestors and descendants. You can set up diagram appearance in "Options" dialog on "Tree" tab (you can get an access to "Options" dialog via "Edit/Options..." menu item). You can change information to show about persons and change colors of tree elements.

To build a tree, you pick a person out in person list, then choose a tree type (ancestors, descendants or complete) in "Pedigree" menu. Also you can use the following main window's toolbar's buttons:

  •  Tree of ancestors
  •  Tree of descendants
  •  Complete tree

After you have created a tree, you can use button to export the tree in one of the following formats: raster format (BMP, JPEG, PNG or GIF) or vector one (EMF). Not every family tree may be saved as a bitmap file, because the latter may not store more than 65k pixels per horizontal line. Vector file does not have such restriction; therefore, it can store trees of any size. Moreover, having a vector file is better for the next image processing in graphical applications and printing.

The following controls are available when you view a diagram:

  • vertical scrolling with mouse wheel;
  • horizontal scrolling with mouse wheel and holding [ Shift ] key;
  • zoom with mouse wheel and holding [ Ctrl ] key;
  • zoom with scale control at top right corner of the window;
  • [ F5 ] key rebuilds the tree;
  • [ F6 ] key marks the currently selected person as a starting point and updates all ancestors and descendants of the starting point;
  • [ F7 ] key recalculates relation degrees (if they are turned on) of the currently selected person;
  • double click on a person to open it in edit mode;
  • click on a person to adjust it with window's center;
  • pan the content while mouse's right button is held;
  • click on icon ("there are additional hidden data") near a person to rebuild the tree relatively to that person (this icon is not exported to image file).

Number of visible generations (of both ancestors and descendants) can be adjusted from one to nine (+ with no limits) by the button with drop down menu on the window's toolbar.

Also you can filter content out with button.

If you have used [ F6 ] key or icon (set new person as a central one for the tree), you can use and buttons to return to the previous or next person.
