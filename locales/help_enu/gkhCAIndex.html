Authenticity index

Authenticity index of information about person is a special indicator that can be evaluated for person entity only.

Each reference to a source in a person entity has specific authenticity level of mention that man in the source. Authenticity level can have the following values:

  • Unreliable acknowledgement or supposed data (authenticity is 25%),
  • Doubtful acknowledgement (authenticity is 50%),
  • Indirect evidences (authenticity is 75%),
  • Direct and primary evidences (authenticity is 100%).

While you are researching and viewing some information, authenticity level shows to you how much you can trust that information. It is better to set "Doubtful acknowledgement" if a source document had blots, was damaged and could not be read with confidence. This can prevent you from false conclusions.

If a reference to a source has no authenticity level, such reference gets "Unreliable acknowledgement or supposed data" by default. Just because mentioning a person in document is the important acknowledgement by itself.

Authenticity index is evaluated automatically for each person basing on authenticity level (as the weighted arithmetic mean). You can view authenticity index in statistics module for analysis of mean values or in tree diagrams for pictorial presentation. The index is evaluated using all references to sources, specified in both person entity and all materials and events.
