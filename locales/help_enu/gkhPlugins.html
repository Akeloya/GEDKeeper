Plug-ins

There are two types of extensions available for GEDKeeper: Lua scripts and plug-ins written in C# or VB.NET.

The following are main reasons to extend an application with plug-in units:

  • User wants to add new functions to application without having to rebuild source code and deploy new release;
  • User wants to add new functions to application but does not have necessary source code.

To create plug-ins, you must use GKCore common library of application's core.
