Multimedia record

Multimedia record stores documents, photos, audio and video records.

There are three ways you can use to store multimedia content in database: store as direct link to file, store as archive and store in "storage". Please read Files storing article to get additional information.

Multimedia records found united directory for entire digitized multimedia content available for user. Any time you can access this directory and view required file. Photos, as a type of multimedia record, can be used as portrait for a person record.

There is one crucial point when you are adding new multimedia record to database: use any external file carefully. You should not add files with size more than 10 MiB using "In archive" storage type. Usually such large files are audio or video content with very poor compression rate. This all increases size of database dramatically. It is recommended to store large content either as link to external file or as "storage" -- special file directory locating in the same folder with database.

"Multimedia" tab on the main working window contains united table with all multimedia content available in database. A record summary to the right of the table shows information about selected record and references made from other records to this one.

See also: Note, Source, Storing multimedia content.
