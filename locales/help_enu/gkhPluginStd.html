Standard plug-ins

There are some plug-in modules available with default SKU:

  • Expression calculator, a standard service plug-in;
  • Calendar, a standard service plug-in;
  • Names reference book, a standard service plug-in;
  • Time line, a standard service plug-in;
  • Stream data input plug-in streamlines mass data input from sources;
  • Pedigree import plug-in adds data from outer text pedigree;
  • Image viewer plug-in shows sources' scanned images and photos;
  • Full-text search plug-in search an entry in database; works alike a Web search engine;
  • "Ancestors circle" (experimental) plug-in shows ancestors diagram on the form of concentric circles;
  • 3D Visualization (experimental) plug-in shows 3D tree diagrams;
  • "Conway Life" game plug-in helps you to relax.

See also: Plug-ins.
