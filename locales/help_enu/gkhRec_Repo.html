Archive record

Archive records combine sources located at one storage place. Archive is either State archives, private archive institution or a private collection, or some specific document storage.

Archive has name, storage unit list, address and other contact information. You can also add some text note about hours of service, for example, and other information.

There is a uniting table on the "Archives" tab of the main working window, which includes all archives available in database. Archive summary to the right of the table shows all information about selected archive and sources that, according to database, are stored in selected archive.

See also: Source, Note, Address.
