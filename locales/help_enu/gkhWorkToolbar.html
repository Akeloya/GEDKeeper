Main toolbar

GEDKeeper's main toolbar is shown on the following picture:

The toolbar has the following buttons:

Database management
 Create new database
 Open existing database
 Save database modifications
Record management (depended on active tab)
 Add new record
 Modify record
 Remove record
 Filter record
Family tree and pedigree ("Persons" tab only)
 Open ancestor tree for selected individual
 Open descendant tree for selected individual
 Open combined tree with ancestors and descendants for selected individual
 Create pedigree for selected person
Other
 Show statistic information for individuals
Navigation
 Move backward to the previous record
 Move forward to the next record
