Tools: pedigrees import

Attention:

  • Import processes pedigrees in text format only.
  • Person must be identified by Konovalov's scheme.
  • Person identifier and its name must be delimited by period (for example, "2-1. Daniel Garner").
  • Name must have the following format:"First_name Patronymic_name Last_name".
  • Person sex is determined basing on typical features of Russian language's woman names, on names reference index, and, in all other cases, on user decision (via prompting).

A recommendation: make several import attempts and check result after each attempt. If you find an error, modify the origin text. For example, frequently birthdays inside a row with name are not marked with "*" symbol. It is impossible to recognize such situations intelligently, because this will increase number of false positives. Moreover, name may contain name variants enclosed in parentheses, without parentheses or with "?" mark. It is impossible to recognize these with a simple enough algorithm. It is better to extract such data, and various date formats, to a text block after the row with name.
