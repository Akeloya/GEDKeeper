About GEDKeeper

GEDKeeper is a free software that manages your personal genealogical database. 'GED' prefix in the product title stands for GEDCOM -- international de facto standard describing how to store and exchange genealogical data and used by many commercial genealogical software. The only specification used by GEDKeeper to store its data is GEDCOM. This extremely advances GEDKeeper compatibility level with other products.

GEDKeeper has very simple and self-explanatory user interface. Therefore for user with any experience working with this application is straightforward. Target audience of the product is most of all Russian users, because GEDKeeper manipulates patronymic names, automatically handles woman last names using specific features of the Russian language, etc.

This is a brief list of GEDKeeper features:

  • Very easy installation process,
  • Full support of GEDCOM standard,
  • No limits for number of persons and generations used,
  • Generates pedigrees painting,
  • Builds several types of genealogical tree,
  • Prints and exports family-trees to images,
  • Quick navigation and management of multimedia content,
  • Puts georeferenced data over Google Maps (© Google).

Author's aim was to create an easy-to-use product and not an overladen all-in-one stuff. The primary goal is to invent fast and effective tool to manage personal genealogical database. A tool that will become an addition to the traditional card index.
