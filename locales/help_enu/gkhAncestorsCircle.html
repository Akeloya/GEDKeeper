Ancestors circle

Ancestors circle is a special-purpose diagram that shows all ancestors of selected person as sectors of concentric circles. Settings for this diagram are available at "Diagrams\Ancestors circle" tab in "Options" window. You can open "Options" dialog using "Edit\Options..." menu. Settings on the specified tab allows you to set up the diagram appearance.

To generate "Ancestors circle" diagram select a target entry in the persons list and then use "Ancestors circle" item in "Pedigree" menu.

The following actions are available while you are viewing the diagram:

  • [ Left ] key updates diagram of father of selected person,
  • [ Right ] key updates diagram of mother of selected person,
  • [ Up ] key increases the number of displayed generations,
  • [ Down ] key decreases the number of displayed generations,
  • [ Backspace ] key returns to a previous person,
  • click a sector to update diagram of selected person,
  • doule click outside the diagram to change the current view mode to ancestors compositional analysis mode or revert the current mode back.

The diagram always shows fixed number of ancestral generations (eight generations).

If you have changed the central person clicking a sector of the diagram, you can use and buttons to switch to a previous or next person.

There is an additional ancestors compositional analysis mode available in "Ancestors circle" diagram: you can double click outside the diagram to conduct ancestor duplication analysis. This analysis changes the diagram as follow: all unique, non-duplicated ancestors have chlorine color, while all ancestors having two or more occurrence have different color (each duplicating ancestor has its own color). This allows you to evaluate genetic variety of ancestors more clearly.
