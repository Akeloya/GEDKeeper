File operations

Create new file

Select "New" menu item in "File" menu or click button on the toolbar.

Open existing file

Select "Open..." menu item in "File" menu or click button on the toolbar.

Save file

Select "Save..." menu item in "File" menu or click button on the toolbar.

File properties

Select "File properties" in "File" menu. Using opened window specify document author. Also, in this window you can view file summary information and file integrity checking report.

Record operation

View record

Find and select a record you need in a list. You can view complete list of record properties to the right of record list. Summary information may contain hyperlinks that make your navigation through records easy.

Add record

Having a tab with list of necessary record type opened, for example -- "Persons", select "Add record" menu item in "Edit" menu or click button. "New person" window is opened (or another one for the respective record type), where you specify everything you know about the object. Then press "Apply" button to save your data. New entry will appear in the record list.

Modify record

Find a record you need on a tab with record list. Select "Modify record" menu item in "Edit" menu or click button.

Every record type has its own editor window:

  • Person editor
  • Family editor
  • Note editor
  • Multimedia content reference editor
  • Source editor
  • Archive editor
  • Group editor
  • Researching editor
  • Task editor
  • Correspondence editor
  • Location editor

Remove record

Find a necessary record on a tab, select it and use "Remove record" menu item in "Edit" menu or click button.
