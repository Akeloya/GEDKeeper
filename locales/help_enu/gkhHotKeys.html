Keyboard shortcuts

The following is list of hotkeys available in GEDKeeper:

Main window
[ Ctrl + N ] Create new databse
[ Ctrl + O ] Open existing database
[ Ctrl + X ] Exit the application
[ Ctrl + F11 ] Open scripts window
[ F1 ] Show help content (this document)
Database window, records tables
[ Ctrl + S ] Save database
[ Ctrl + I ] Add new record
[ Ctrl + Enter ] Modify a record
[ Ctrl + L ] Delete a record
[ Ctrl + A ] View ancestors tree
[ Ctrl + D ] View descendants tree
[ Ctrl + P ] Create pedigree according to the d'Abbe-ville scheme
[ Ctrl + K ] Create pedigree according to the Konovalov scheme
[ Ctrl + M ] Open maps
[ Ctrl + T ] Open statistics
[ Ctrl + F ] Filter data
Tree diagram window
[ F5 ] Rebuild the tree
[ F6 ] Make selected person record as a start point and rebuild ancestors and descendants of the selected person
[ F7 ] When showing of relation degrees is enabled: rebuild relation degrees for the selected person
