Task record

Task stores information about investigation into specific person, family or source. For this purpose, task has goal, reference to person or family, task priority and due date. You can also add text notes to a task to describe search progress.

You group task by search directions using research records.

There is a uniting table on the "Tasks" tab of the main working window, which includes all tasks available in database. Task summary to the right of the table shows all information about selected task and list of research records that link the selected one.

See also: Person, Family, Source, Note, Research.
