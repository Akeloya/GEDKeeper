How to build GEDKeeper

Microsoft Windows

Dependencies

The following is a list of required software to build and test:

  • Microsoft .NET Framework 4.0.30319
  • Microsoft Visual Studio 2017 or later
  • Nullsoft Scriptable Install System (NSIS) 3.04
  • 7-zip
  • NUnit 2.6.4 (unit testing framework for .NET)

Execution

Run the executable build_all_mswin.x86.cmd from the command line.

Optionally use the parameter "Release" for a release build (the default is a debug build). Testing is not done with a release build.
